# NBA Analytics Dashboard

A modern NBA analytics web application built with Flask, Python, Chart.js, and the NBA API.

This project lets users search NBA players, view season and career statistics, explore interactive charts, and compare players side by side.

## Features

- Search NBA players
- Smart search results for duplicate names
- Player headshots
- Team logos
- Full team names
- Season averages
- Career totals
- Season-by-season tables
- Interactive PPG charts
- Trending Players section
- Compare Players page
- Winner highlighting for stat comparisons
- ESPN-inspired dark UI

## Tech Stack

- Python
- Flask
- nba_api
- Pandas
- HTML
- CSS
- Jinja
- Chart.js
- Git/GitHub
